# 🎬 电影推荐站

一个功能丰富的电影推荐网站，支持分类筛选、搜索、排序，还有精美的动效和用户互动功能。

## ✨ 功能特性

- 📋 **分类筛选**：热门推荐、剧情佳作、奇幻冒险、动画精选
- 🔍 **搜索功能**：按电影名称和描述搜索
- 📊 **内容排序**：按评分、年份、名称排序
- 🎨 **精美动效**：滚动淡入、悬浮放大、平滑过渡
- 👆 **用户互动**：点赞、收藏、评论（数据存储在 localStorage）
- 🖼️ **Lightbox**：图片点击放大查看，支持左右切换
- 📱 **响应式设计**：完美适配移动端和桌面端
- 📷 **AI 生成图片**：所有电影海报都由 AI 生成

## 🚀 使用方法

### 本地运行

直接用浏览器打开 `index.html` 即可。

### 部署到 GitHub Pages

1. 确保代码已推送到 GitHub
2. 进入仓库 Settings
3. 在 Pages 页面，Source 选择 "Deploy from a branch"
4. Branch 选择 `main` 和 `/ (root)`
5. 等待部署，访问 `https://z1b211.github.io/movie-site`
